# Smite QL API

## Pre-Requisites

- Node
- NVM

## Installation

```
nvm use
npm ci
npm start
```

## Versioning

#### Deleting a broken version

_Deleting a remote tag is for when you need to unpublish a version update because it's bad_

```
# tagname is string like "v1.8.1"
git push --delete origin <tagname>
```
